# 컴퓨터 그래픽스 팀 프로젝트

## **3D Interactive Tower Defense**

## React + Vite + Three.js / React Three Fiber
 
### 4팀

# **1\. 프로젝트 개요**

## **1.1 주제 및 선정 이유**

본 프로젝트는 React + Vite 환경에서 React Three Fiber(R3F)를 활용한 3D 인터랙티브 타워 디펜스 게임을 구현한다. 타워 디펜스 장르는 그리드 기반 맵 관리, 적 AI 경로 탐색, 실시간 전투 로직, 파티클 이펙트 등 컴퓨터 그래픽스 핵심 요소를 자연스럽게 포함하기 때문에 과제 요구사항을 유기적으로 충족할 수 있다.

단순한 그래픽 시각화에 그치지 않고 실제로 플레이 가능한 게임을 완성함으로써, 사용자 인터랙션·애니메이션·조명·카메라 제어 등 모든 필수 구현 요소를 하나의 완결된 결과물에 통합한다.

## **1.2 핵심 컨셉**

- 탑뷰(Top-Down) 시점의 3D 전장 위에서 타워를 배치해 적의 침입을 막는 전략 게임
- 모든 오브젝트(타워, 적, 투사체, 맵 타일)는 Three.js Geometry + 커스텀 Material로 구현
- Bloom 포스트 프로세싱 + 파티클 시스템으로 전투 이펙트 강화
- OrbitControls로 자유 카메라 조작 지원

# **4\. 게임 시스템 설계**

## **4.1 맵 구조**

16×10 그리드 배열로 맵을 정의한다. 0은 빈 땅(타워 설치 가능), 1은 적 경로(설치 불가)로 구분하며, 각 셀은 BoxGeometry 1개로 렌더링된다.

## **4.2 적(Enemy) 이동 로직**

- waypoints\[\] 배열에 경로 좌표를 하드코딩
- 매 프레임 currentWaypoint를 향해 position.lerp() 이동
- 웨이브 번호에 따라 속도, 체력, 수량 증가
- drei의 &lt;Html&gt;으로 체력바 3D 공간에 표시

## **4.3 타워(Tower) 공격 로직**

- 매 프레임 주변 적과 distance() 계산 → 사정거리 내 타겟 탐지
- 타겟 방향으로 포신 lookAt() 회전
- 공격 쿨타임(cooldown)마다 투사체(Projectile) 스폰
- 투사체는 타겟 위치로 선형 이동, 명중 시 파티클 이펙트 + 체력 감소

## **4.4 타워 종류**
